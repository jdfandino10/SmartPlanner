SmartPlanner API!

API REST de la aplicación SmartPlanner

Verbo

URL

Descripción

Parámetros

GET

/users

Da todos los usuarios

-

GET

/users?username=default

Da el usuario con el username default (con arreglo ids tarea)

default: nombre del usuario

GET

/users/:id/hmks?category=c&order=o

Da las tareas del usuario con id :id segun los parametros, es decir, las tareas de categoría c ordenadas por o.

c:

-history[orden cronológico, valor por defecto]

-finished

-not_finished

-not_started

o:

-priorized[valor por defecto]

-date

POST

/users/:id/hmks

Crea una nueva tarea al usuario con id :id

body:{

[objeto tarea]

{"name": "Nombre de la tarea", "description": "Descripción de la tarea", "estimated_time": 3600000, "limit_date": 1598413880298, "done_percentage": 1, "importance": 2}

}

PUT

/users/:id/hmks/:id_h

Modifica una nueva tarea de id :id_h. Solo si esta tarea está pendiente.

body:{

[objeto tarea]

{"name": "Nuevo nombre de la tarea", "description": "Nueva descripción de la tarea", "estimated_time": 3600000, "limit_date": 1598413880298, "done_percentage": 1, "importance": 2}

}

DELETE

/users/:id/hmks/:id_h

Elimina la tarea con id :id_h

  

PUT

/users/:id

Modifica al usuario con id :id

body:{

[objeto usuario]

{ "user_name": "Nombre usuario", "email": "correo@correo.com", "subscribed": "true"}

}
